What is HTML?

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

Basic Tools for HTML

In order to write HTML you will need:

  • Notepad on windows
  • TextEdit on Mac
  • or Notepad++
    • These tools will be the foundation place to write your HTML.

Why Learn HTML?

  • Competitive advantage as a skill
  • Improves critical thinking
